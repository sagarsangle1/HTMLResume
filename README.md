# HTML Resume – Single Page Website

This repository contains a **single-page resume website built using pure HTML** as part of an assignment.  
The goal of this project is to demonstrate proper use of **HTML structure, semantics, readability, and layout**, without using any CSS.

---

## 📌 Assignment Requirements

- Build a **single-page resume website** using **HTML only**
- No usage of CSS
- Follow the provided sample layout
- Use proper HTML tags:
  - Headings
  - Paragraphs
  - Lists
  - Tables
  - Sections
- Include the following sections:
  - Header / Name
  - About Me
  - Skills
  - Experience
  - Projects
  - Education
  - Contact
  - Achievements
